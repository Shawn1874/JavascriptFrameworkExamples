Setup 
	1. run 'npm install mocha chai'
	2. run 'npm install mocha -g' to install it globally
		a. Must also ensure that the global npm directory is on the path
		b. %APPDATA%\npm
	3. run 'npm test' or 'mocha Tests/QuickStart'
		

Documentation Links
	1. Chai - https://www.chaijs.com/
	2. Mocha - https://mochajs.org/
